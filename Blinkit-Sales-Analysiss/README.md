# Blinkit Sales Analysis Dashboard

## Project Overview
This project analyses Blinkit grocery sales performance using Power BI to understand sales trends, product performance, and outlet distribution.

The dashboard helps identify key business insights such as top-performing product categories, outlet sales contribution, and customer rating trends.

## Tools Used
Power BI  
Excel  
Data Visualisation  

## Dataset
The dataset contains information about grocery items sold across different Blinkit outlets including:
- Item type
- Outlet location
- Outlet size
- Item rating
- Sales performance

## Key KPIs
Total Sales: $1.20M  
Average Sales: $141  
Total Number of Items: 8523  
Average Rating: 3.9  

## Dashboard Features
- Sales analysis by item category
- Sales distribution by outlet size
- Outlet establishment trend analysis
- Sales contribution by outlet location tier
- Product fat content sales analysis
- Outlet type performance comparison

## Key Insights
- Tier 3 outlets generate the highest revenue among all outlet locations.
- Supermarket Type1 contributes the majority of overall sales.
- Low-fat items account for a significant portion of total sales.
- Sales performance varies significantly across item categories.
