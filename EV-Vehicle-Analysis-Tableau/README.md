# Electric Vehicle Data Analysis (Tableau)

## Project Overview
This project analyzes Electric Vehicle (EV) adoption trends using Tableau. The goal is to understand EV growth, manufacturer dominance, and geographic distribution.

## Tools Used
Tableau
Excel

## Key KPIs
- Total Vehicles
- Average Electric Range
- Total Battery Electric Vehicles (BEV)
- Total Plug-in Hybrid Electric Vehicles (PHEV)

## Dashboard Insights
- EV adoption has grown rapidly after 2020
- Tesla dominates EV market share
- Certain states show significantly higher EV adoption
- Battery Electric Vehicles account for majority of EV registrations
